# Sliding Stage OPM 프로파일 분석 로직 — 기술 문의 결과 보고

> **작성일**: 2026-04-02
> **목적**: 자체 개발 프로그램의 분석 알고리즘을 연구소 기준 Tool과 비교·검증하여, 차이점을 파악하고 수정 방향을 정리
> **참조**: PMS Q&A 4249

---

## 1. 요약

자체 개발 프로그램의 4대 지표 (Rep. Max, Rep. 1σ, OPM Max, OPM 1σ)를 연구소 기준 Tool 결과와 비교한 결과,
**Flatten(평탄화) 방식, 이상치 처리, 일부 지표 공식**에서 차이가 확인되어 수정이 필요합니다.

| 구분 | 일치 여부 | 원인 |
|------|-----------|------|
| Rep. Max | ✅ 거의 일치 (오차 0.001 nm) | Flatten 방식 차이만 존재 |
| Rep. 1σ | ❌ 불일치 (−37% ~ −67%) | **공식 자체가 다름** |
| OPM Max | ⚠️ 부분 일치 (±2~5%) | Flatten 방식 + 이상치 처리 차이 |
| OPM 1σ | ❌ 완전 불일치 (37배 차이) | **지표 정의 자체가 다름** |

---

## 2. 항목별 비교표

| 항목 | 자체 개발 (수정 전) | 연구소 기준 Tool | 차이점 요약 |
|------|---------------------|-----------------|-------------|
| **Flatten 방식** | Rep: 2차 다항식 / OPM: 1차 다항식, **전체 데이터로 fitting** | **공통 1차 다항식**, **양끝 1%만으로 fitting** 후 전체에 적용 | Fitting에 사용하는 데이터 구간이 다름 |
| **이상치 처리** | 없음 (모든 픽셀 사용) | 동일 위치에서 측정 간 편차가 큰 **비정상 픽셀을 자동 제외** | 이상치 자동 제외 유무 |
| **Rep. Max** | 픽셀별 편차 중 최대값 | 동일 (유효 픽셀만 대상) | 이상치 제외 여부만 차이 |
| **Rep. 1σ** | 픽셀별 편차의 **표준편차** | 픽셀별 반복 표준편차의 **RMS** 집계 | **계산 공식이 다름** |
| **OPM Max** | 전체 픽셀의 Max−Min 중 최대 | **유효 픽셀**의 Max−Min 중 최대 | 이상치 제외 여부 차이 |
| **OPM 1σ** | 5회 OPM 값의 **표준편차** (변동성) | 전체 측정 높이값의 **RMS** (형상 크기) | **지표 의미 자체가 다름** |

## 3. 핵심 차이 3가지 (비기술 요약)

### ① Flatten(평탄화) — "어디를 기준으로 수평을 맞추느냐"

프로파일 데이터는 기울어져 있어 분석 전 수평 보정(Flatten)이 필요합니다.

| | 자체 개발 | 기준 Tool |
|--|----------|-----------|
| 기준점 | 전체 데이터를 사용해 기울기 계산 | **양쪽 끝 1%만** 사용해 기울기 계산 |
| 비유 | 자 전체를 바닥에 대고 수평 맞춤 | 자의 **양쪽 끝만** 바닥에 맞추고 가운데는 그대로 |

→ 가운데에 Bow(휨)가 있을 경우, 전체 fitting은 Bow를 일부 제거해버리지만, 양끝 fitting은 Bow를 보존합니다.

### ② 이상치 제외 — "비정상 데이터를 걸러내느냐"

같은 위치를 5번 측정했을 때, 특정 픽셀에서 1번만 비정상적으로 큰 값이 나올 수 있습니다.

| | 자체 개발 | 기준 Tool |
|--|----------|-----------|
| 처리 | 모든 데이터 그대로 사용 | 편차가 큰 상위 x% 픽셀을 **자동 제외** |
| 영향 | 이상치 1개가 전체 결과를 왜곡 | 이상치 영향 최소화 |

→ 실제 사례: 3_RT 위치의 Sample18에서 비정상값(148nm, 정상 97~100nm) 1개로 인해 Rep. Max가 6.3배 과대 산출됨.

### ③ OPM 1σ 정의 — "무엇을 측정하느냐"

| | 자체 개발 | 기준 Tool |
|--|----------|-----------|
| 측정 대상 | 5회 측정값이 얼마나 **흔들리는지** (반복 변동성) | 프로파일 형상이 얼마나 **큰지** (Bow 크기) |
| 결과 범위 | ~2 nm (매우 작음) | ~68 nm (프로파일 크기 반영) |

→ 같은 이름이지만 완전히 다른 의미의 지표였음.
